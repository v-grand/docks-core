# Обзор Платформы

Эта страница предоставляет общий обзор архитектуры платформы, описывая ее ключевые компоненты и их назначение.

## template-platform

**Точка сборки и запуска всей платформы.**

`template-platform` является центральным компонентом, который объединяет все остальные сервисы, обеспечивая их взаимодействие и развертывание. Он служит основой для всей экосистемы, предоставляя общие конфигурации, инструменты сборки и механизмы запуска.

## Сервисы Платформы

Ниже представлен список основных сервисов, составляющих платформу:

### [template-user-service](template-user-service.md)

**Аутентификация, роли, сессии.**
Сервис управления пользователями, отвечающий за регистрацию, аутентификацию, управление ролями и сессиями пользователей.

### [template-wallet-service](template-wallet-service.md)

**Кошельки, подписи, Web3-интеграция.**
Сервис для управления цифровыми кошельками, обработки криптографических подписей и интеграции с Web3-технологиями.

### [template-payment-service](template-payment-service.md)

**Платежи, шлюзы, Web3/Fiat.**
Сервис обработки платежей, поддерживающий различные платежные шлюзы, а также операции с Web3-активами и традиционными (Fiat) валютами.

### [template-order-service](template-order-service.md)

**Заказы, статусы, корзина.**
Сервис управления заказами, отслеживания их статусов и работы с корзиной покупок.

### [template-product-service](template-product-service.md)

**Каталог, товары, категории.**
Сервис для управления каталогом товаров, их описаниями, категориями и атрибутами.

### [template-discount-service](template-discount-service.md)

**Скидки, акции, купоны.**
Сервис для управления скидками, акциями и купонами, применяемыми к товарам и заказам.
